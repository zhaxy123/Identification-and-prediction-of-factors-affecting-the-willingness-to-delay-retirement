# 个体延迟退休意愿的影响因素识别与预测研究

### 项目简介
本项目基于2020年中国健康与养老追踪调查（CHARLS）数据，构建“延迟退休意愿”二分类变量。为识别影响因素并精准预测个体延迟退休意愿，本文首先从个人基本特征、经济因素与健康状况三个维度构建分析框架，运用逻辑回归模型识别各因素对延迟退休意愿的影响方向与强度；在此基础上，进一步引入决策树与随机森林模型，通过预测性能比较，筛选出能够精准识别政策目标响应群体的最优模型。
  
### 研究目标
- 探索影响个体延迟退休意愿的主要影响因素；
- 构建逻辑回归、决策树以及随机森林预测模型并评估对比其性能，筛选出能够精准识别政策目标响应群体的最优模型；
- 为公共政策制定提供数据支持；

### 数据集
- 数据来源：2020年中国健康与养老追踪调查（CHARLS）数据库 https://charls.pku.edu.cn/ 
- 样本量：2096条记录 （通过筛出异常和缺失数据）
- 目标变量：retdelw（是否愿意延迟退休）
- 特征数：9个（性别、年龄、教育、婚姻状况、收入、户口类型、健康水平、子辈经济支持、在学孩子数）
